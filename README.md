# 🚢 Titanic Data Cleaning & Preprocessing Project

## 📌 Overview

This project focuses on **data cleaning and preprocessing** using the famous Titanic dataset. The goal is to transform raw, unclean data into a structured and analysis-ready format.

---

## 🎯 Objective

To clean and prepare the dataset by handling missing values, removing duplicates, converting data types, and renaming columns for better readability.

---

## 🏢 Internship

This project was completed as part of an internship at **Synent Technologies**, where I worked on real-world data science tasks and gained hands-on experience in data preprocessing.

---

## 📂 Dataset

* **Name:** Titanic Dataset
* **Source:** Kaggle
* Contains passenger details such as age, gender, class, fare, and survival status.

---

## 🛠️ Tasks Performed

### ✅ 1. Handling Missing Values

* Filled missing values using appropriate methods:

  * Mean / Median for numerical columns
  * Mode or placeholders for categorical columns

### ✅ 2. Removing Duplicates

* Identified and removed duplicate records to ensure data integrity

### ✅ 3. Data Type Conversion

* Converted columns to correct data types (e.g., age → float, survived → integer)

### ✅ 4. Renaming Columns

* Renamed columns to make them more readable and consistent
